
## Installation information
### Download prebuilt version
Just head to the releases tab and download the .jar file. Put that into your Minecraft mods folder and you are done

### Build it yourself
Just use a 21 JDK like Eclipse Temurin 21.0.8 with gradle. Then build with "./grade build" and drag the built jar file from /build/libs/ into your Minecraft mods folder
